# Проект: Лендинг-страница для магазина часов
Суть проекта: Создание современного, адаптивного лендинга для презентации ассортимента часов, с возможностью выбора товара и оформления заказа.
Что удалось сделать:
-	Воссоздан дизайн и верстка всех секций сайта в соответствии с макетом
-	Создана компонентная архитектура сайта
-	Реализовано динамическое отображение каталога товаров из Vuex
-	Внедрена интерактивность выбора товара и модальное окно для оформления заказа
-	Настроена адаптивность под мобильные устройства с помощью медиа-запросов 
-	Стиль выполнен на SCSS с использованием переменных и вложенности для удобства поддержки
  
Стек:
- Vue.js,
- Vuex,
- HTML,
- CSS,
- SCSS,
- JavaScript
  
  # Установка и запуск проекта
Cкачайте зависимости с помощью команды в консоли:
    - `nmp install`

   После установки зависимостей, запустите проект с помощью команды:
- `npm run serve`
